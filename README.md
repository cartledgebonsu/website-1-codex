# HOMEY Next.js Homepage

This is a Next.js App Router homepage prototype for HOMEY CONSTRUCTION COMPANY LIMITED.

## Built From

- `黑方-Codex外贸独立站信息填写表.xlsx`
- HOMEY product/logo/application images supplied by the user
- HOMEY SEO keyword system and competitor analysis from the current thread

## Run Locally

Install dependencies first:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Notes

- The current Codex runtime has Node.js but no `npm`/`npx`, so the server could not be started here.
- The page uses real HOMEY-provided assets under `public/assets`.
- The inquiry form is front-end interactive and shows the submitted success message. It is not connected to an email or CRM backend yet.
- Social links included: Instagram, LinkedIn and Facebook.
